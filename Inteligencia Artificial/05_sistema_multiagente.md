# Sistemas Multiagentes
- dividir para conquistar (vários agentes realizando tarefas paralelamente)
- faz parte da subárea de IA conhecida como IA Distribuida - inspirada nas organizações/comunidades de insetos (swarm intelligence)
  - as técnicas de resolução de problemas são divididas/repartidas entre agentes numa sociedade.
  - dentro da computação um agente pode ser uma entidade de hardware ou uma entidade de software distribuídos geograficamente (técnicas de comunicação de dados - sockets, multicast, ...)
 - Aplicação de sistemas multiagentes == EQUIPES/TIMES
  - Sistemas Pervasivos/Ubíquos -> automação residencial e industrial
  - Características:
    - autonomia ➜  threads
    - proatividade ➜  threads + ifs
    - adaptação/flexibilidade ➜ try-catch | sobrecarga de predicados
    - habilidade social == comunicação (comando/ordem; pergunta; plantio de crença)
        - socket
        - diretivas de comunicação (.broadcast; .tell; .send)
- Há inúmeras arquiteturas
    - Imperativa ➜ OO e programação procedural
    - BDI ➜ Belief, Desire and Intention
        - Interpretador JASON para a linguagem AgentSpeak(L) (baseada no Prolog)
        - belief = crença == fato 
        - desire = desejo == plano em tese/teoria/latente
        - intention = intenção = plano em execução
